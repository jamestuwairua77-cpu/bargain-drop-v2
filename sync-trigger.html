🔧 Bargain Drop Sync Panel

Product Sync

Pull new/updated products from Shopify and rebuild the website data files.

Shopify Webhooks

Registered webhook endpoints. Product creates/updates trigger automatic rebuilds.

Manual Command

Sync trigger:
curl -X POST https://bargain-drop.online/api/product-sync-webhook
-H "X-Shopify-Topic: products/create"
-d '{"id":1,"title":"test"}'
